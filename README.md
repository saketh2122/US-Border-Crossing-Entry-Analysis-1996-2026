🚀 US Border Crossing Entry Analysis (1996–2026)

📌 Project Overview

The **US Border Crossing Entry Analysis** is a data analytics project designed to analyze and visualize border crossing patterns from **1996 to 2026**. This project leverages **Excel and Power BI dashboards** to uncover meaningful insights that help in understanding cross-border traffic trends and volume reporting.

---

🎯 Objectives

* Analyze border crossing data to identify patterns and trends  
* Detect high-traffic ports, regions, and time periods  
* Perform data cleaning and preprocessing for accurate analysis  
* Build an interactive dashboard for easy visualization  
* Generate insights for better understanding of transportation modes  

---

📊 Features

* 📈 Year-wise and date-wise trend analysis  
* 🗺 Region-wise and border-wise distribution (US-Canada, US-Mexico)  
* ⏰ Time-based crossing pattern insights  
* 🔍 Measure-wise breakdown (Vehicles, Buses, Pedestrians, Trucks, etc.)  
* 📊 KPI metrics such as Total Value and Active Ports  
* 📊 Interactive dashboard using Power BI  

---

🛠️ Tech Stack

* **Data Processing:** Microsoft Excel  
* **Visualization Tool:** Power BI  
* **Techniques:** Data Cleaning, Data Transformation, Dashboard Design  

---

📂 Dataset

The dataset contains US border crossing records from **1996 to 2026**, including details such as:

* Date of crossing  
* Port name and location  
* Border type (US-Canada / US-Mexico)  
* State  
* Measure (type of transportation)  
* Total crossing value  
* Latitude and longitude  

---

⚙️ Workflow

1. **Data Collection** – Imported dataset from Excel files  
2. **Data Cleaning** – Handled missing values and inconsistencies  
3. **Exploratory Data Analysis (EDA)** – Identified trends and patterns  
4. **Dashboard Development** – Built interactive dashboards in Power BI  
5. **Visualization** – Created charts, maps, and KPI indicators  

---

📈 Key Insights

* Certain ports consistently show higher crossing volumes  
* Personal vehicles contribute the highest share of traffic  
* Border activity varies significantly across years and regions  
* Geographic analysis highlights high-density crossing areas  
* Trends can help in understanding transportation and movement patterns  

---

🚀 How to Run the Project

1. Clone the repository  
2. Open the Excel dataset  
3. Open the Power BI (.pbix) file  
4. Explore the interactive dashboard
